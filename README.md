A graphics showcasing application built with Python's turtle module that leverages algorithmic patterns and randomized RGB color schemes to generate various geometric designs.

Features

Procedural Shape Drawing: Cycles through nested loops to construct multi-sided regular polygons from triangles up to decagons.
Random Walk Simulation: Projects a continuous mathematical path across four strict perpendicular axes using persistent heading changes.
Spirograph Engine: Shifts orbital heading angles by granular parameters to create complex overlapping circle arrays.
Dynamic Colorization: Queries custom RGB channels sequentially to ensure completely unique vector color palettes during calculation loops.
